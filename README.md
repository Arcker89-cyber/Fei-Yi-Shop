# Fei Yi Shop v3 — Stock Management System

## 🆕 ฟีเจอร์ใหม่

### 1. หน้า Home แสดงสินค้าแนะนำ
- แสดงเฉพาะสินค้าที่ทำเครื่องหมาย "แนะนำ" (featured)
- สินค้าแนะนำจะมี badge สีทอง ⭐

### 2. แยกหมวดหมู่สินค้า
- **⚡ เครื่องมือไฟฟ้า** (electric)
- **🚗 เครื่องมือซ่อมรถ** (auto)
- **🔧 เครื่องมือช่างทั่วไป** (hand)
- กดแท็บเพื่อกรองสินค้าตามหมวดหมู่

### 3. ระบบจัดการสินค้า (Admin Panel)
- ➕ เพิ่มสินค้าใหม่
- ✏️ แก้ไขสินค้า
- 🗑️ ลบสินค้า
- 📊 ดูสถิติสต๊อก
- 🔄 รีเซ็ตเป็นค่าเริ่มต้น

## 🚀 วิธีใช้งาน

1. เปิด `index.html` - หน้าร้าน
2. เปิด `admin.html` - จัดการสินค้า (เพิ่ม/แก้ไข/ลบ)

---

## 📦 ระบบสต๊อก

| สถานะ | เงื่อนไข | สี |
|--------|---------|-----|
| มีสินค้า | stock > 10 | 🟢 เขียว |
| ใกล้หมด | stock 1-10 | 🟡 เหลือง |
| หมดสต๊อก | stock = 0 | 🔴 แดง |

---

## ⚙️ วิธีเพิ่มสินค้าใหม่

1. เปิด `admin.html`
2. กดปุ่ม "➕ เพิ่มสินค้าใหม่"
3. กรอกข้อมูล:
   - ชื่อสินค้า
   - ราคา
   - สต๊อก
   - หมวดหมู่
   - URL รูปภาพ
   - คำอธิบาย
   - ✅ ติ๊กถูก "สินค้าแนะนำ" ถ้าต้องการให้แสดงหน้าแรก
4. กด "💾 บันทึก"

---

## 💾 การเก็บข้อมูล

ข้อมูลสินค้าเก็บใน **localStorage** ของ browser
- สินค้า: `feiyi_products_v3`
- ตะกร้า: `feiyi_cart_v3`

⚠️ ถ้าล้าง browser data จะสูญเสียสินค้าที่เพิ่มเอง (กดรีเซ็ตเพื่อเอาค่าเริ่มต้นกลับมา)

---

## 🔧 Console API (สำหรับ Developer)

```javascript
// ดูสินค้าทั้งหมด
FeiyiApp.products.getAll()

// เพิ่มสินค้าใหม่
FeiyiApp.products.addProduct({
  title: "สินค้าใหม่",
  price: 999,
  stock: 50,
  category: "hand",
  img: "https://...",
  desc: "คำอธิบาย",
  featured: true
})

// ดูสถิติ
FeiyiApp.products.getStats()

// รีเซ็ตข้อมูล
FeiyiApp.products.resetToDefault()
```
